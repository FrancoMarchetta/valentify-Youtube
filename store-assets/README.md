# Store Assets

This directory contains promotional assets for Chrome Web Store.

## Required Assets

### Screenshots (Required - at least 1)
- Size: 1280x800 or 640x400 pixels
- Format: PNG or JPEG
- Show the extension in action

**Recommendations:**
1. YouTube homepage with modified thumbnails
2. Search results page with MrBeast faces
3. Settings popup

### Small Promo Tile (Recommended)
- Size: 440x280 pixels
- Format: PNG or JPEG
- Use for featured placement

### Large Promo Tile (Optional)
- Marquee: 1400x560 pixels
- Format: PNG
- For prominent store placement

## How to Create Screenshots

1. Load the extension in Chrome
2. Go to YouTube
3. Press F12 to open DevTools
4. Click on device toolbar (mobile icon)
5. Set dimensions to 1280x800
6. Take screenshots with Windows Snipping Tool or similar
7. Save them here as `screenshot-1.png`, `screenshot-2.png`, etc.
